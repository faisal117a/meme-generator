InstantDB Schema Sync Guide

Important: Your InstantDB dashboard only shows $files and $users because the custom schema hasn't been synced yet. Follow these steps to create the memes and upvotes tables.

Method 1: Using InstantDB CLI (Recommended)

Step 1: Install InstantDB CLI

npm install -g @instantdb/cli

Step 2: Login to InstantDB

instant login

Step 3: Sync Schema

Navigate to your project directory and run:

instant sync --app-id 2cc27c95-cf7d-4fe7-9fe4-4c56c35cda96

Or if you have the schema file in the current directory:

instant sync instant.schema.js --app-id 2cc27c95-cf7d-4fe7-9fe4-4c56c35cda96

Method 2: Manual Setup via Dashboard

Step 1: Go to InstantDB Dashboard

Visit: https://www.instantdb.com/dashboard

Select your app: 2cc27c95-cf7d-4fe7-9fe4-4c56c35cda96

Step 2: Create "memes" Entity

In the Schema section, create a new entity called memes with these fields:

- imageData: string (required) - title: string (optional) - authorId: string (required) - authorEmail: string (required) - createdAt: number (required)

Permissions:

  • Read: Everyone (public)
  • Create: Authenticated users only
  • Update: Author only (ctx.auth.id === entity.authorId)
  • Delete: Author only (ctx.auth.id === entity.authorId)

Step 3: Create "upvotes" Entity

Create another entity called upvotes with these fields:

- memeId: string (required) - userId: string (required) - createdAt: number (required)

Permissions:

  • Read: Everyone (public)
  • Create: Authenticated users only
  • Update: Disabled
  • Delete: User can delete own upvotes (ctx.auth.id === entity.userId)

Method 3: Use the Schema File Directly

Copy Schema Content

Your schema file (instant.schema.js) contains the correct structure. You can:

  1. Copy the content from instant.schema.js
  2. Go to InstantDB Dashboard â†’ Schema
  3. Click "Import Schema" or paste the schema definition

After Syncing

Once the schema is synced, you should see:

  • $files (system table)
  • $users (system table)
  • memes (your custom table)
  • upvotes (your custom table)

Then your app will be able to store memes and upvotes!

Note: If you're using the InstantDB React SDK, make sure your schema file is properly configured. The schema should automatically sync when you deploy or use the CLI tool.
